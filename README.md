# fwmsoftware.com redesign — static site package (Harbor & Coral)

This folder is the ready-to-publish redesign of fwmsoftware.com: three static pages (`index.html`, `versions.html`, `contact.html`) plus one shared stylesheet (`styles.css`) built directly from the Harbor & Coral design tokens, with no build step, no frameworks, and no external assets beyond the Outfit font from Google Fonts. To publish, upload the four files to the web host's document root, replacing the existing files (the demo forms are static-safe — they open a prefilled email to support@fwmsoftware.com, so no server-side form handler is needed). Intentionally not included: the blog/news section, the downloads page, and the per-version detail pages from the old site — those are either retiring or pending a separate decision, and nothing here links to them.
